Question-1: What are the different ways to select an element in the DOM?

We can select elements in the DOM using getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll methods.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The innerHTML property sets or gets the HTML content of an element, innerText sets or gets only the visible text, and textContent sets or gets all the text including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation means using a single event listener on a parent element to handle events on its child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where an event starts from the target element and moves up to its parent elements in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create elements using document.createElement(), add them using appendChild() or append(), and remove them using removeChild() or remove().
